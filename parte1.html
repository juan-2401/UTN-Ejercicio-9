Ejercicio 9 - Parte 1

Consigna

  1. Definir dos constantes (variables) llamadas usuario y password y asignar el valor administrador y 1234 respectivamente.
  2. Preguntar al usuario que ingresa al sitio a través de dos prompts el usuario y la contraseña del sistema.
  3. Si el usuario y la contraseña coinciden con las del sistema (variable usuario y password) mostrar en el navegador un mensaje de bienvenida
  4. Si no coinciden mostrar usuario o contraseña incorrectos.
